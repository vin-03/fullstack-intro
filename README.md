# Firebase CRUD Table

A simple web application built with Next.js and Firebase that allows users to manage data in a table format.

## Description

The app supports some of CRUD functionality:
- Add new entries  
- Delete records  
- Real-time data updates via Firebase

## Technologies Used

- Next.js  
- React  
- JavaScript  
- Firebase (Firestore)
